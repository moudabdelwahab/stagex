توثيق API منصة مدعوم

مرحباً بك في توثيق API مدعوم. تتيح لك واجهة برمجة التطبيقات الخاصة بنا ربط قدرات الذكاء الاصطناعي والذاكرة الخاصة بمدعوم في تطبيقاتك ومواقعك الخاصة بسهولة.

Base URL: https://api.mad3oom.online/v1

المصادقة (Authentication)

يتم استخدام مفتاح API (Bearer Token) للمصادقة على جميع الطلبات. يجب إرسال المفتاح في ترويسة الطلب (Headers) كالتالي:

HTTP
Authorization: Bearer YOUR_API_KEY

نقاط الاتصال (Endpoints)

1. إرسال رسالة للشات

إرسال رسالة إلى البوت والحصول على رد ذكي بناءً على سياق الذاكرة.

POST /chat/send
المعلمة (Parameter)النوعالوصف
messagestringنص الرسالة المرسلة
session_idstringمعرف الجلسة للحفاظ على سياق المحادثة

أمثلة برمجية

JavaScript (Fetch API)

JS
fetch('https://api.mad3oom.online/v1/chat/send', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        message: 'مرحباً، كيف حالك؟',
        session_id: 'user_123'
    })
})
.then(res => res.json())
.then(data => console.log(data.reply));

Python (Requests)

Python
import requests

url = "https://api.mad3oom.online/v1/chat/send"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
payload = {
    "message": "مرحباً، كيف حالك؟",
    "session_id": "user_123"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json()['reply'])

PHP (cURL)

PHP
$ch = curl_init("https://api.mad3oom.online/v1/chat/send");
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer YOUR_API_KEY',
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    "message" => "مرحباً",
    "session_id" => "user_123"
]));
$response = curl_exec($ch);
echo json_decode($response)->reply;
