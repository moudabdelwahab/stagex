🔧 الإصلاح الأول: إظهار الإعدادات المتقدمة

تم إضافة قسم الإعدادات المتقدمة إلى كل بطاقة مفتاح API. يتضمن هذا القسم ثلاثة خيارات رئيسية:

  • Allowed domains / IPs: تحديد النطاقات وعناوين IP المسموحة
  • Platform binding: ربط المفتاح بمنصة معينة (Web, Mobile, Server)
  • HMAC: تفعيل توثيق HMAC للطلبات
.advanced-settings { background: #fafafa; padding: 1.5rem; border-radius: 8px; display: block !important; border: 1px solid #eee; }

🎨 الإصلاح الثاني: إصلاح تداخل القائمة الجانبية

تم تصحيح CSS للقائمة الجانبية في صفحة التوثيق لضمان عدم تداخلها مع المحتوى الرئيسي. تم استخدام الحسابات الديناميكية للعرض:

.main-content { margin-right: var(--sidebar-width); width: calc(100% - var(--sidebar-width)); box-sizing: border-box; padding: 3rem; }
